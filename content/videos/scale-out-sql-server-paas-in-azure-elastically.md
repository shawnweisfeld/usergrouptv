## Abstract:

<p>You have heard all the hype how the cloud can help you scale your databases horizontally, providing you limitless capacity, but how do you do that? In this presentation we will take a look at the Elastic Database tools. We will focus on the key use cases for shard map management, data dependent routing and multi-shard querying.</p>
