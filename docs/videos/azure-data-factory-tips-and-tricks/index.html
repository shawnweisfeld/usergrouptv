Azure Data Factory Tips and Tricks

Abstract:

* Zipping a large file in an Azure Storage Account so your download is smaller * Disable an Activity in Azure Data Factory * Do a partial Publish in Azure Data Factory instead of the whole Publish All I'll start out with a quick overview of what each tip is. Then I'll go through a longer step by step process of setting up each tip.
