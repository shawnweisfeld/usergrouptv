Azure SQL Database: Where Does My Data Belong?

Abstract:

Today, we have three different options for storing our relational data. There is Azure SQL VM, Azure SQL Database and Azure SQL Managed Instances. We will talk about the pros and cons of each so you can make the right decision for your data. You will learn common pitfalls between migrating and managing your data between these three offerings. You will also learn the differences between these three options for high availability, disaster recovery, performance and security.
