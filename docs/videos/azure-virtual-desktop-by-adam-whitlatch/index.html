Azure Virtual Desktop

Abstract:

Azure Virtual Desktop (aka. Windows Virtual Desktop) is a desktop app virtualization service that runs in Azure. Come learn how Windows Virtual Desktop can help you deploy and scale your Windows Desktops and Apps in Azure in minutes all while allowing your employees to work in a secure manner wherever they are? We’ll discuss the current capabilities, newly released features, and Partner Integration such as Citrix, and Horizon.
  • Technical overview
  • Windows 10 Multi-user
  • WVD Global Service
  • New Features & Updates in the latest release
  • Integration into other Azure services
  • Azure Portal Integration
  • Partnership with Citrix / VMware
  • 3rd Party integrators and tools
  • Q&A
