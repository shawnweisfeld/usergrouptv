Basic UI Design for iPhone, Android and Windows Phone

Abstract:

So you’ve decided that building a native UI will deliver the best experience for your users. That’s great for customers, but now what? You may be familiar with design on one of the top platforms, but how do the other guys do it? This talk will show you how to build a very simple calculator on Android, iPhone and Windows Phone. We’ll see the tools, the markup and the code that actually makes these apps work. We’ll even explain the differences along the way! Finally, we’ll talk briefly about some sites and local user groups you can get involved in to improve your mobile development skills and expand to new platforms.
