Blockchain Programming in iOS Using Swift

Abstract:

Blockchain is a revolutionary technology, which powers the cryptocurrencies including Bitcoin, Litecoin, Ether etc. Apart from the cryptocurrencies, Blockchain can be applied to many other fields including retail, finance, medial, media etc. In this session you are going to learn the principles behind the Blockchain technology including proof of work, consensus, blocks, transactions and more. Next, we are going to implement a simple Blockchain using Swift language. Finally, we will take our Blockchain to the cloud using server side Swift framework. By the end of this session you will have a better understanding of the concepts behind Blockchain and how you can implement it using any programming language.
