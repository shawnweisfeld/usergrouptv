Building cross-platform mobile applications using jQuery Mobile

Abstract:

In this session, we'll discuss how to write a functional, touch-friendly web application using jQuery Mobile. We'll walk through basic pages, layouts, toolbars, navigation, list views, events, form controls, and transition effects.
