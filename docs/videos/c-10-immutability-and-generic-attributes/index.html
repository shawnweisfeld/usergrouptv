C# 10: Immutability and Generic Attributes

Abstract:

We have a brand new version of C#, and it is awesome. We see continued deeper support for Immutability in C#, and a feature I’ve desperately wanted for years, Generic Attributes! Come join me as we explore these new features with hands on code examples.
