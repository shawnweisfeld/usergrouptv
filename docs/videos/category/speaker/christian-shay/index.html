Christian Shay

Christian Shay is a product manager in the .NET technologies group at Oracle. As a .NET and Oracle evangelist, he speaks to user groups, at Microsoft events and at Oracle Openworld events worldwide. Prior to this role, he was a software engineer at Oracle. Read his article on the Oracle Data Provider for .NET, Managed Driver in the March issue of Oracle Magazine (http://www.oracle.com/technetwork/issue-archive/2014/14-mar/o24odp-2147205.html).
