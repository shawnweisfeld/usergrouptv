Stephen W. Thomas

Stephen W. Thomas is an independent consultant specializing in Azure Hybrid Integration solutions using Microsoft BizTalk Server, Azure Logic Apps, and other Microsoft technologies. He has almost 20 years experience consulting with clients all over the world and has been recognized as a Microsoft Most Valuable Professional (MVP) in BizTalk Server / Microsoft Integration / Azure.

Stephen run the BizTalk community site http://www.BizTalkGurus.com which offers a forum, numerous BizTalk and Azure samples, how-to videos, and his blog. He has presented at several Microsoft TechEd events, multiple Service Oriented Architecture Conferences and various user groups throughout the world. He is also an author with Pluralsight and currently have five courses available online covering both BizTalk Server and Azure Logic Apps.
