Deploy All The Games!

Abstract:

Come see how you can use the Unity3D toolset to build and deploy your game to ALL THE DEVICES!. New devices do not need ports from scratch anymore, we are in the 21st Century. We will explore how to deal with differing form factors, pixel densities, graphics support, input systems, and other multi-platform madness.It can be done. It doesn't have to be hard.
