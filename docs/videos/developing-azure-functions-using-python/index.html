Developing Azure Functions using Python

Abstract:

Join us for a technical talk where we delve into the world of Azure Functions, a serverless compute service that allows developers to build and deploy event-driven applications with ease. Through the session, we will do a deep dive on Azure Functions, common use cases, how to get started, and demonstrate an end to end example using Azure Functions in Python to build an AI application.
