Entity Framework – The Right Way

Abstract:

If you have ever worked with Entity Framework, even if you’re doing Entity Framework Code First, you are almost certainly doing it wrong. While Entity Framework is a wonderful framework, it doesn’t automatically force you into the correct patterns of use – especially if you’re interested in isolation and testability. This talk will review the concepts of both the Repository Pattern and the Specification Pattern and demonstrate how to implement them in Entity Framework.
