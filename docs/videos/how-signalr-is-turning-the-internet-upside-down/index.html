How SignalR is Turning the Internet Upside Down

Abstract:

ASP.NET SignalR is a somewhat new library for ASP.NET developers that makes it incredibly simple to add real-time web functionality to web applications. This session will provide an introduction to SignalR. We will take a look at how to develop with SignalR, some of the underlying implementations that SignalR uses, and what common and enterprise problems that SignalR can help solve.
