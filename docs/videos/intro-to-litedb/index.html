Intro to LiteDB

Abstract:

Have you ever needed a simple database for a web or windows application? What if I could demonstrate a database that: * Merely needs a small .NET DLL to run * Simply installable with NuGet * Compatible with .NET 4.5 and .NET Standard 2.0 * Delivers very fast performance * Uses a single data file * Has LINQ query support * Supports encrypted data files * Is a NoSQL database * Easily allows you to run a database in memory * FREE, including for commercial use In my action pack code filled presentation I will reveal all that and more. Don't miss it!
