Introduction to Direct Lake datasets in Fabric

Abstract:

Microsoft announced a new and innovative Power BI dataset storage mode - Direct Lake in Microsoft Fabric. This new mode promises to completely change the way Power BI datasets are developed and consumed by combining the best of import and Direct Query modes. In this session, Sandeep Pawar will go over the details and show how it works and help you get started the right way to make the most of it.
