Introduction to Xamarin.iOS and Xamarin.Android

Abstract:

Learn the basics of building an application in Xamarin.iOS and Xamarin.Android and how you can share code between the two platforms. On each platform we will cover application life cycle, view management, view layout, and event handling as well as platform-specific issues like view controller life cycle (iOS) and Activity/Fragment life cycle (Android).

Speakers:

  • Adam Kemp

    6 Recordings

    Adam Kemp is a Xamarin MVP and a Principal Software Architect at National Instruments with 3 years of Xamarin experience.
