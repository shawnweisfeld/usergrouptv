Keeping Up With JavaScript

Abstract:

How can we "level up" our software development abilities without leaving our familiar, comfortable languages? We can try using an unusual programming paradigm, or introducing artificial constraints which force us to see things through a different lens. Brandon Byars and Pete Hodgson will discuss "leveling up" by using JavaScript in a different way, with some practical examples: not using the `this` keyword, and applying the Functional Reactive paradigm to JavaScript development.
