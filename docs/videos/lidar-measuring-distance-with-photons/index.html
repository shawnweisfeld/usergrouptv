LiDAR - Measuring Distance with Photons

Abstract:

Laser ranging finding technologies (LIDAR) have been around for a while. LIDAR enjoys an ever increasing number of applications, and an ever broadening range of sensor options. These days, it's easy to find inexpensive LIDAR sensors and integrate them with Arduino and Raspberry Pi. This session will give a high level intro to LIDAR, and give several live demonstrations using maker-grade hardware.
