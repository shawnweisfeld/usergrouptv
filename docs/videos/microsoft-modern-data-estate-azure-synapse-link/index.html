Microsoft Modern Data Estate & Azure Synapse Link

Abstract:

Session focus to provide an overview of Microsoft modern data estate and the use of the Azure Synapse Link technology to reduce their time to insight over their data. This framework and tools help to bridge the gap between transactional systems and analytics systems satisfying the need to not negatively impact transactional performance while also providing analytics and warehouse platforms with near-real time visibility to changes in transactional systems. Azure Synapse link services will help to make data pipelines more flexible while reducing customer TCO to maintain data feeds.
