Microsoft Purview: Unified Data Gov Made Easy and Extensible!

Abstract:

Do you work with data? Does someone need to know where it came from, who owns it and what exactly Key1, id_vat_no, and cust_spec_type means? Then you will want to learn about the Data Discovery, Data Classification, and Data Governance features in Microsoft Purview. This session will introduce you to the product, what it can do, and then dive into how you can extend it to make it even more useful.
