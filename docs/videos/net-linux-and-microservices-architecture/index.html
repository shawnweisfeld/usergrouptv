.NET, Linux and Microservices Architecture

Abstract:

Microsoft’s move to make .NET open source could not have come at a better time. As the world explores and moves to Microservices Architecture (MSA) and Linux containers, .NET stands poised to not only join the fray, but to excel with high-performance services. In this fast-paced session, you will see .NET code (C# and F#) be turned into microservices, running in Red Hat’s OpenShift PaaS. Linux, .NET, docker, kubernetes and MSA will all be tied together in this peek into the future. To finish up, we’ll demonstrate two deployment methods: Blue/Green and Canary, and show how uninterrupted deployments are possible.
