Overview of Azure Storage

Abstract:

Join Sr. Program Manager Shawn Weisfeld from the Azure Storage team for tour of Azure Storage. The Azure Storage platform is Microsoft's cloud storage solution for modern data storage scenarios. Azure Storage offers highly available, massively scalable, durable, and secure storage for a variety of data objects in the cloud, including object, file, disk, queue, and table storage. In this session learn about your storage options in Azure.
