Pattern-Driven Insight: Integrating Neo4j with Power BI

Abstract:

This session will introduce you to the world of Graph Databases by using Neo4j to find hidden patterns in Stock Market Price History and then use Power BI to visualize and explore those patterns to gain insights. Bryant will walk you through Machine Learning algorithms he used to identify and generate hidden patterns. He will also teach you a new Power Query technique he developed to combine applied steps into a single query step allowing you to make reusable Template Queries with input variables.
