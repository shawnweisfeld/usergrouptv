Portable Libraries: Write Once, Utilize Everywhere

Abstract:

It is closer that you think. / WinRT: Welcome to the v1 worldâ€¦ again. Yes, it is possible to develop, using C# or VB, a core set of libraries that can be leveraged across WinRT (Windows 8 Metro), WinForms, WPF, Silverlight, ASP.NET, Windows Phone, Mono (Linux, Mac, Android, iOS). This can be done using Visual Studio 2010 (an extension) and Visual Studio 2012 (built in). In this talk, one of many methods will demonstrate how the core of an application is built and leveraged across Windows Phone, Silverlight, WinForms and WinRT. Additionally, we will spend a significant amount of time covering some of the specifics developing for WinRT from a Windows Phone 7 and Silverlight perspective.
