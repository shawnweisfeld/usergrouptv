PostgreSQL in Azure

Abstract:

Interested in running Postgres, join this session to learn about the different Postgres offerings Microsoft has running in Azure as well as on-premises. In this session we will cover Postgres Single Server, Flex Server, Hyperscale running the Citus extension in Azure as well as running Postgres Hyperscale (Citus) on premises in your data center.
