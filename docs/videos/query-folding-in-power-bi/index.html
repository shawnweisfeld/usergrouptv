Query Folding in Power BI

Abstract:

Did you know Power BI can generate native SQL queries which can speed up refreshes? In this session, we will look at how query folding works, when it works/doesn't work and some hidden tricks to get it to work. The techniques covered will be helpful even when working with non-folding data sources.
