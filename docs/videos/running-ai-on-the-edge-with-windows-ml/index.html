Running AI on the Edge with Windows ML

Abstract:

If you're a C# developer and want to use trained machine learning model in your Windows apps, this session is for you. This session is going to show you how AI and ML can help you solve real problems in your own apps and services and how easy it is to get started with Machine Learning today in your own apps. You will learn the basics of machine learning, how to use existing models and services in your windows apps, and how to get started with creating your own simple models. A step-by-step process will be presented. Windows ML makes it easy for you to leverage the infinite possibilities of AI by establishing a simple process to integrate models with your application.
