The Varying Ways to Develop Mobile Apps in 2017

Abstract:

We'll cover all the ways to code mobile apps including Native, Hybrid, & Cross-Platform apps like React Native, NativeScript, Xamarin, Unity and more! We'll compare/contrast the language choices, level of native knowledge needed and developer experience to help you make a more informed choice on your next Mobile App project.
