Tutorial on Building Scalable Web Applications in the Cloud

Abstract:

This is a walk-through on how to architect, build, and deploy web applications in the Cloud for people who have not built cloud applications before. This covers a few general cloud architectures, avoiding bottlenecks, and the nuts and bolts of configuring and deploying cloud servers.

Speakers:

  • Jonathan Bartlett

    1 Recording

    Jonathan Bartlett is the Director of Technology at New Medio, where he leads a team of developers building web and mobile applications for customers across the globe. Jonathan is the author of several books on computer programming, including "Programming from the Ground Up", "New Programmers Start Here", and "Building Scalable Web Applications Using the Cloud".
