Using Jupyter Notebook With Power BI

Abstract:

Did you know that you can not only embed Power BI reports in Jupyter notebooks but now you can also generate auto-created reports in notebooks? In this session, we will look at how to achieve this using Python and explore practical use cases, especially for exploratory data analysis. Although some familiarity with Python will be helpful, it's not a prerequisite to attend this session.
