Using T-SQL to find differences between tables

Abstract:

When you have to find differences and changes in your tables, there are many methods and tools to use.We'll cover a range of t-sql code, from basic to complex, and settings to find these differences and changes: Brute force - write it all out, Change Data Capture, T-sql Merge statement, And more.
