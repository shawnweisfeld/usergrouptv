What's New in Visual Studio 2012 for Web Developers

Abstract:

Building websites has become more and more important for most businesses worldwide, and with the introduction of HTML 5 we now have more possibilities than ever. Join us on a journey through the features of the upcoming version of Microsoft ASP.NET and Visual Studio to get a sneak peek on the new abilities and tools needed to build modern websites.
